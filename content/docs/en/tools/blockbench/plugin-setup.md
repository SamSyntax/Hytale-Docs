# Hytale Plugin Setup

Configure the Hytale Blockbench plugin.

## Features

- `.blockymodel` format
- `.blockyanim` animations
- Hytale-specific settings
- Bone hierarchy tools

## Create New Model

1. File > New
2. Select "Hytale Model"
3. Start modeling
