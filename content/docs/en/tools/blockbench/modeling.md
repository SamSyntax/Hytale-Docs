# Modeling Guide

Create 3D models for Hytale.

## Getting Started

1. Create new Hytale Model
2. Add cubes for geometry
3. Set up bone hierarchy
4. Add textures
5. Export as `.blockymodel`

## Tips

- Face models toward +Z
- Use proper bone naming
- Minimize polygon count
