# Blockbench Installation

Install Blockbench for 3D modeling.

## Download

Get Blockbench from [blockbench.net](https://blockbench.net)

## Install Hytale Plugin

1. Open Blockbench
2. File > Plugins
3. Search "Hytale"
4. Click Install

## Verify

Create a new Hytale Model project to confirm.
