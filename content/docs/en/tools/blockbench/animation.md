# Animation Guide

Animate your models.

## Creating Animations

1. Open Animation panel
2. Create new animation
3. Set keyframes
4. Use graph editor for curves
5. Export as `.blockyanim`

## Best Practices

- Start with idle animation
- Keep movements smooth
- Test in-game frequently
