# API Endpoints

Official Hytale API endpoints.

## Base URL

```
https://hytale.com/api
```

## Endpoints

### Blog Posts

```
GET /blog/post/published
```

Returns published blog posts.

### Post by Slug

```
GET /blog/post/slug/{slug}
```

Returns specific post.
