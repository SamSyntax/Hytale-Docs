# Controls

Master Hytale's controls for an optimal experience.

## Movement

| Action | Default Key |
|--------|-------------|
| Forward | Z / W |
| Backward | S |
| Left | Q / A |
| Right | D |
| Jump | Space |
| Sprint | Shift (hold) |
| Crouch | Ctrl |
| Walk | Alt (hold) |

## Combat

| Action | Default Key |
|--------|-------------|
| Primary attack | Left Click |
| Secondary attack | Right Click |
| Block / Parry | Right Click (with shield) |
| Special attack | F |
| Dodge | Double-tap direction |
| Lock target | Tab |
| Switch target | Scroll Wheel |

## Inventory and Interface

| Action | Default Key |
|--------|-------------|
| Open inventory | E |
| Action bar 1-9 | Keys 1-9 |
| Drop item | G |
| Use item | Right Click |
| Map | M |
| Quest log | J |
| Pause menu | Escape |
| Chat | Enter / T |

## Building

| Action | Default Key |
|--------|-------------|
| Place block | Right Click |
| Destroy block | Left Click (hold) |
| Select block | Scroll Wheel |
| Copy block (pick) | Middle Click |
| Rotate | R |
| Build mode | B |

## Camera

| Action | Default Key |
|--------|-------------|
| Look around | Mouse |
| Zoom | Scroll Wheel (build mode) |
| First person view | F5 |
| Third person view | F5 (x2) |
| Screenshot | F2 |
| Photo mode | F4 |

## Social

| Action | Default Key |
|--------|-------------|
| Player list | Tab (hold) |
| Voice chat (push-to-talk) | V |
| Emotes | Y |
| Social menu | O |

## Customization

All keys can be reconfigured in:

**Menu > Settings > Controls**

### Control Profiles

Hytale allows you to create multiple profiles:
- One for combat
- One for building
- One for exploration

### Controller Support

Hytale supports Xbox and PlayStation controllers:

| Action | Xbox | PlayStation |
|--------|------|-------------|
| Movement | Left Stick | Left Stick |
| Camera | Right Stick | Right Stick |
| Attack | RT | R2 |
| Block | LT | L2 |
| Jump | A | X |
| Interact | X | Square |
| Inventory | Y | Triangle |
| Sprint | LS (click) | L3 |

## Useful Shortcuts

### Advanced Combat

- **Quick combo**: Left click x3 rapidly
- **Charged attack**: Hold left click
- **Counter-attack**: Block then immediate left click
- **Roll**: Sprint + Crouch

### Quick Building

- **Quick placement**: Hold right click while moving
- **Quick selection**: Hold Ctrl + Scroll Wheel
- **Copy structure**: Ctrl + C (in build mode)
- **Paste structure**: Ctrl + V
