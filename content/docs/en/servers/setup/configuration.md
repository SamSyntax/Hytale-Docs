# Server Configuration

Configure your server.

## server.properties

```properties
server-name=My Server
port=25565
max-players=50
view-distance=10
```

## Key Settings

| Setting | Description |
|---------|-------------|
| `server-name` | Display name |
| `port` | Network port |
| `max-players` | Player limit |
| `view-distance` | Render distance |
