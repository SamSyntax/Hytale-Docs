# Server Requirements

Hardware and software requirements.

## Software

- Java 21+
- Linux, Windows, or macOS

## Hardware (Minimum)

| Component | Spec |
|-----------|------|
| CPU | 4 cores |
| RAM | 6-8 GB |
| Storage | 20 GB SSD |

## Hardware (Recommended)

| Component | Spec |
|-----------|------|
| CPU | 6+ cores |
| RAM | 12-16 GB |
| Storage | 50 GB NVMe |
