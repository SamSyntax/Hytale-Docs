# Server Installation

Install your Hytale server.

## Quick Start

```bash
# Download server
# (from hytale.com)

# Create directory
mkdir hytale-server
cd hytale-server

# Start server
java -Xms4G -Xmx8G -jar hytale-server.jar
```

## First Run

The server will:
1. Generate config files
2. Accept EULA
3. Generate world
4. Start listening
