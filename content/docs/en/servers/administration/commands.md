# Server Commands

Console and in-game commands.

## Common Commands

| Command | Description |
|---------|-------------|
| `/stop` | Stop server |
| `/kick <player>` | Kick player |
| `/ban <player>` | Ban player |
| `/op <player>` | Give operator |
