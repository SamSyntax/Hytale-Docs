# Whitelist

Control who can join your server.

## Enable Whitelist

```properties
# server.properties
whitelist=true
```

## Commands

| Command | Description |
|---------|-------------|
| `/whitelist add <player>` | Add player |
| `/whitelist remove <player>` | Remove player |
| `/whitelist list` | Show all |
