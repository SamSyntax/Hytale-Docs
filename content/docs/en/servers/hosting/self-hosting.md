# Self-Hosting Guide

Host on your own hardware.

## Requirements

- Static IP or dynamic DNS
- Port forwarding capability
- 24/7 uptime (optional)

## Steps

1. Install Java 21+
2. Download server files
3. Configure firewall
4. Port forward 25565
5. Start server
