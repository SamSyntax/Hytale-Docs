# Installation du serveur

Installez votre serveur Hytale.

## Démarrage rapide

```bash
# Télécharger le serveur
# (depuis hytale.com)

# Créer le répertoire
mkdir hytale-server
cd hytale-server

# Démarrer le serveur
java -Xms4G -Xmx8G -jar hytale-server.jar
```

## Premier lancement

Le serveur va :
1. Générer les fichiers de configuration
2. Accepter l'EULA
3. Générer le monde
4. Commencer à écouter les connexions
