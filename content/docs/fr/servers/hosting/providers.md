# Hébergeurs

Services d'hébergement de jeux pour Hytale.

## Options

| Type | Avantages | Inconvénients |
|------|------|------|
| Hébergeur de jeux | Configuration facile | Moins de contrôle |
| VPS | Bon équilibre | Configuration moyenne |
| Serveur dédié | Contrôle total | Plus complexe |

*Les recommandations d'hébergeurs seront ajoutées au lancement des services.*
