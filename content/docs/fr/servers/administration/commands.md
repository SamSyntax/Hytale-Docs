# Commandes du serveur

Commandes console et en jeu.

## Commandes courantes

| Commande | Description |
|---------|-------------|
| `/stop` | Arrêter le serveur |
| `/kick <player>` | Expulser un joueur |
| `/ban <player>` | Bannir un joueur |
| `/op <player>` | Donner les droits opérateur |
