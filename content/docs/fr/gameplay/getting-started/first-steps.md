# Premiers Pas sur Hytale

Bienvenue sur Orbis ! Ce guide vous accompagnera dans vos premières heures de jeu.

## Lancement du Jeu

### Configuration Requise

| Composant | Minimum | Recommandé |
|-----------|---------|------------|
| OS | Windows 10 64-bit | Windows 11 64-bit |
| Processeur | Intel i5-4460 / AMD FX-6300 | Intel i7-8700 / AMD Ryzen 5 3600 |
| RAM | 8 Go | 16 Go |
| GPU | GTX 960 / RX 560 | RTX 2060 / RX 5700 |
| Stockage | 20 Go SSD | 50 Go SSD |

### Premier Lancement

1. Téléchargez Hytale depuis [hytale.com](https://hytale.com)
2. Installez le jeu
3. Connectez-vous avec votre compte Hytale
4. Choisissez votre mode de jeu

## Création de Personnage

Avant de commencer, personnalisez votre avatar :

- **Apparence** - Visage, coiffure, couleur de peau
- **Vêtements** - Tenue de départ
- **Nom** - Votre identité sur Orbis

## Votre Première Zone : Central Grasslands

Vous commencez dans les **Prairies Centrales** (Central Grasslands), le cœur d'Orbis.

### Objectifs Initiaux

1. **Récoltez des ressources de base**
   - Bois (frappez les arbres)
   - Pierre (minez les rochers)
   - Fibres végétales (herbes hautes)

2. **Fabriquez vos premiers outils**
   - Hache en bois
   - Pioche en bois
   - Épée basique

3. **Construisez un abri**
   - Trouvez un emplacement sûr
   - Posez des murs et un toit
   - Ajoutez une porte

4. **Survivez à la nuit**
   - Les créatures hostiles apparaissent la nuit
   - Restez à l'intérieur ou préparez-vous au combat

## Conseils pour Débuter

### Exploration

- Repérez les **points d'intérêt** (POI) sur le terrain
- Les colonnes de fumée indiquent des camps
- Les tours en ruine cachent souvent des trésors

### Combat

- Utilisez l'**endurance** avec sagesse
- Apprenez les patterns d'attaque des ennemis
- Bloquez et esquivez avant de contre-attaquer

### Ressources

- Récoltez tout ce que vous trouvez au début
- Les coffres dans le monde contiennent des objets utiles
- Parlez aux PNJ pour des quêtes et conseils

## Prochaines Étapes

Une fois les bases maîtrisées :

1. Explorez les **donjons** pour du meilleur équipement
2. Apprenez la **cuisine** pour des buffs
3. Développez vos **compétences de combat**
4. Découvrez de nouvelles **régions**

:::tip Sauvegarde
Hytale sauvegarde automatiquement votre progression. En mode solo, vous pouvez aussi créer des sauvegardes manuelles.
:::
