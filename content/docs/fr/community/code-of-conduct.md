# Code de Conduite

## Notre Engagement

Nous nous engageons à fournir un environnement accueillant et inclusif pour tous.

## Normes

### Comportement Attendu

- Être respectueux et inclusif
- Donner et recevoir des critiques constructives
- Se concentrer sur ce qui est le mieux pour la communauté
- Faire preuve d'empathie envers les autres

### Comportement Inacceptable

- Harcèlement ou discrimination
- Trolling ou commentaires insultants
- Attaques personnelles ou politiques
- Publication d'informations privées d'autrui

## Application

Les violations peuvent entraîner :
1. Avertissement
2. Bannissement temporaire
3. Bannissement permanent

## Signalement

Signalez les problèmes aux mainteneurs via :
- GitHub Issues (pour les affaires publiques)
- Message direct aux mainteneurs (pour les affaires privées)

## Attribution

Ce Code de Conduite est adapté du [Contributor Covenant](https://www.contributor-covenant.org/).
