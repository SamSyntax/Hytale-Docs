# Animations

Animez vos modèles en utilisant Blockbench.

## Format

Les animations utilisent des fichiers `.blockyanim`.

## Créer des Animations

1. Configurez la hiérarchie des os
2. Créez l'animation dans Blockbench
3. Définissez les keyframes
4. Utilisez l'éditeur de courbes pour les transitions
5. Exportez en `.blockyanim`

Voir le [Guide d'Animation](/docs/tools/blockbench/animation).
