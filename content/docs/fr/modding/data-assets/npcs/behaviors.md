# Comportements des PNJ

Configurez comment les PNJ se comportent dans le monde.

## Types de Comportement

- **Idle** - Reste immobile/se promène
- **Aggressive** - Attaque les joueurs
- **Passive** - Fuit les menaces
- **Neutral** - Attaque si provoqué
