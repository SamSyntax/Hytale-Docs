# Créer des PNJ

Créez des PNJ personnalisés pour votre monde.

## Définition de Base d'un PNJ

```json
{
  "id": "my_mod:custom_creature",
  "displayName": "Créature Personnalisée",
  "health": 20,
  "model": "models/custom_creature.blockymodel"
}
```
