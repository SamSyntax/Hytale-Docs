# Créer des Objets

Apprenez à créer des objets personnalisés.

## Définition de Base d'un Objet

```json
{
  "id": "my_mod:ruby_sword",
  "displayName": "Épée de Rubis",
  "type": "weapon",
  "properties": {
    "damage": 10,
    "durability": 500
  }
}
```
