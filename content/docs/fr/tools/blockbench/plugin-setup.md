# Configuration du plugin Hytale

Configurer le plugin Hytale pour Blockbench.

## Fonctionnalites

- Format `.blockymodel`
- Animations `.blockyanim`
- Parametres specifiques a Hytale
- Outils de hierarchie des os

## Creer un nouveau modele

1. File > New
2. Selectionner "Hytale Model"
3. Commencer la modelisation
