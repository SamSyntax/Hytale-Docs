# Guide de modelisation

Creer des modeles 3D pour Hytale.

## Pour commencer

1. Creer un nouveau Hytale Model
2. Ajouter des cubes pour la geometrie
3. Configurer la hierarchie des os
4. Ajouter des textures
5. Exporter en `.blockymodel`

## Conseils

- Orienter les modeles vers +Z
- Utiliser un nommage correct des os
- Minimiser le nombre de polygones
