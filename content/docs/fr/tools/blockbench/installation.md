# Installation de Blockbench

Installer Blockbench pour la modelisation 3D.

## Telechargement

Obtenez Blockbench depuis [blockbench.net](https://blockbench.net)

## Installer le plugin Hytale

1. Ouvrir Blockbench
2. File > Plugins
3. Rechercher "Hytale"
4. Cliquer sur Install

## Verification

Creez un nouveau projet Hytale Model pour confirmer.
