# Guide d'animation

Animer vos modeles.

## Creer des animations

1. Ouvrir le panneau Animation
2. Creer une nouvelle animation
3. Definir les images cles
4. Utiliser l'editeur de courbes pour les transitions
5. Exporter en `.blockyanim`

## Bonnes pratiques

- Commencer par l'animation idle
- Garder les mouvements fluides
- Tester frequemment en jeu
