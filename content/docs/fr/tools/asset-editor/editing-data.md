# Edition des assets de donnees

Utiliser l'editeur d'assets pour modifier le contenu du jeu.

## Types supportes

- Blocs
- Objets
- PNJ basiques

## Rechargement en direct

Les changements s'appliquent instantanement en jeu.

## Limitations

Certains types d'assets ne sont pas encore supportes.
