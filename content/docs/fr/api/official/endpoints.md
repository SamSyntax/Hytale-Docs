# Endpoints de l'API

Endpoints de l'API officielle Hytale.

## URL de base

```
https://hytale.com/api
```

## Endpoints

### Articles de blog

```
GET /blog/post/published
```

Retourne les articles de blog publies.

### Article par slug

```
GET /blog/post/slug/{slug}
```

Retourne un article specifique.
