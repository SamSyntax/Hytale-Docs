# Authentification API

Les endpoints publics ne necessitent pas d'authentification.

## Endpoints publics

- `/blog/post/published`
- `/blog/post/slug/{slug}`
- `/jobs`

## Avenir

Des endpoints authentifies pourraient etre ajoutes pour les operateurs de serveurs.
